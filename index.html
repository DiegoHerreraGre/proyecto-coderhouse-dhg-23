El servicio educacional

El tipo de servicio educacional ofrecido es de estilo tutoría privada. Tendrá distintos planes y podrá ser contratado más abajo. El pago por servicio se hace directamente a través del tutor y el profesor usando los datos bancarios del receptor y con abono previa la clase. La estructura de las clases son dictaminadas en base al currículo nacional de Chile, pero con las interpretaciones sobre qué es más importante por parte del profesor. El estudiante y el profesor se deben poner de acuerdo sobre cuándo hacer las clases y se dispondrá de un horario público en este mismo sitio web donde se podrá ver qué disponibilidades hay. Sobre las tutorías privadas para PAES, estas constarán de un refuerzo paralelo al pre universitario, pero nunca será un sustituto por lo que se recomienda que se siga haciendo uso de este tipo de instancias para que su hijo/a pueda tener el mejor puntaje competitivo.

Plan Focalizado

El plan base es una sola clase por semana. Este plan cuesta el abono de 13 mil pesos por cada clase de una hora. Es ideal para refuerzos particulares o para preparación de una prueba. No se recomienda este plan para trabajos de largo y mediano plazo.

Plan Premium

Este plan incluye dos clases por semana de una duración de 1 hora. En este plan se ve principalmente como un rercurso para mejorar las notas y un mejor entendimiento sobre la materia en general. El enfoque de este plan es más que nada poder establecer una dinámica de estudio autónoma al estudiante que varíe del plan de estudio colegial y que lo prepare para los desafíos futuros.

Plan Deluxe

Este plan incluye un intensivo de tres clases por semanas, además de la posibilidad de hacer clases en la semana. Aquí se busca el aumento drástico de las notas para obtener resultados a mediano plazo, sobre todo para un estudiante que busque mejorar su promedio pensando en la universidad. Se recomienda para estudiantes que estén buscando resultados altos en la PAES.

Plan PAES

Este plan es un intensivo para estudiantes del cuarto medio que busquen grandes resultados en la PAES, sobre todo con la meta de entrar en universidades de renombre. Para ello, este plan solo busca obtener resultados competitivos de los estudiantes, no el aprendizaje. Este plan es recomendado para personas que ya hayan contratado alguno de los dos planes anteriores, para que haya un seguimiento de cómo el estudiante ha logrado mejorar con el tiempo.
